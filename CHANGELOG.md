# Changelog

## 1.0 - Initial public release

- Initial public release of SCS-CN Builder.
- Added editable CN lookup table with default CN2 values.
- Added CN2 raster generation for average antecedent moisture conditions.
- Added automatic CN3 calculation for high antecedent moisture conditions.
- Added basin-level zonal statistics for CN2 and CN3.
- Added quality-control checks.
- Added HTML report generation.
